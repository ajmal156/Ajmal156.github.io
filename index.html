auto h(Node* n, Node* goal) {
  int dx = abs(n->x - goal->x);
  int dy = abs(n->y - goal->y);
  return 1*(dx+dy) + (1.414-2)*min(dx,dy);
}
Multi-Agent Path Finding (MAPF) with CBS
2026
Core research project implementing the Conflict-Based Search (CBS) algorithm for optimal multi-agent path planning. CBS builds a constraint tree where each node represents a set of constraints per agent, resolving vertex and edge conflicts between agents. Supports warehouse simulation scenarios with 10+ concurrent agents on dynamic grid maps.
